Connection Help

How to configure email notifications.

App Password Setup:

  1. Go to your Google Account settings.
  2. Select Security, turn on 2-Step Verification.
  3. Click on App Passwords.
  4. Select Mail and generate a password.
  5. Copy the 16-character code and paste it here.

Google OAuth2 Setup:

  1. Go to Google Cloud Console.
  2. Create a project and enable the Gmail API.
  3. Configure the OAuth Consent Screen (External/Internal) and add scope: https://www.googleapis.com/auth/gmail.send.
  4. Go to Credentials, click Create Credentials -> OAuth client ID (Web Application).
  5. Add the Google Redirect URI shown on the left to your Authorized Redirect URIs.
  6. Click Connect Gmail Account to securely authorize and link your Google Account.
